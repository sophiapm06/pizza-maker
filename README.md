# Sophia's Pizzariea ‚Äî Mini Web Game

A fun, drag-and-drop pizza-making game with an intro that says: **"welcomes to sophias pizzariea"**. Click the Play button to start, match the toppings order, and rack up a score before the timer ends.

## Files
- `index.html` ‚Äî markup for the game
- `style.css` ‚Äî styling
- `script.js` ‚Äî game logic

## How to run
1. Download these files to a folder.
2. Double-click `index.html` (or open it with any modern browser like Edge/Chrome/Firefox/Safari).
3. Optional: serve the folder with a local web server (e.g., `python -m http.server`) for best results.

## Controls
- Drag toppings onto the pizza, or click/tap a topping to auto-place.
- Keyboard shortcuts: `1`=Cheese, `2`=Pepperoni, `3`=Mushroom, `4`=Bell Pepper, `5`=Onion, `6`=Olive, `7`=Pineapple.
- **Bake & Serve** to submit; **Undo** to remove last; **Clear** to reset the pizza.

## Customize
- Edit the `toppings` array in `script.js` to add/remove toppings or change counts.
- Tweak timer/score logic near the top of `script.js`.
- Adjust colors and sizes in `style.css`.
